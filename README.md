# AI Filmmaking Prompts

A collection of prompts and guides for cinematic AI video generation,
built from hands-on experience.

## What's inside

Prompt guides covering shot composition, camera movement, lighting,
color grading, and camera aesthetics — everything needed to get
cinematic results out of AI video models.

## How to use

Each guide includes prompt examples and notes on what works and
what doesn't. Start with the basics and build from there.
